## Code

```py title="fibo.py"
def fib(n: int) -> int:
    if n < 0:
        raise ValueError("n must be non-negative")
    if n < 2:
        return n
    return fib(n - 1) + fib(n - 2)
```

```haskell title="fibo.hs"
fib :: Int -> Integer
fib n
  | n < 0     = error "n must be non-negative Int"
  | otherwise = fibs !! n
  where
    fibs = 0 : 1 : zipWith (+) fibs (tail fibs)

main :: IO ()
main = print (fib 10)
```
